Mblog

mblog(mtons blog)是一个用Java实现的多人轻博客. 使用Mysql数据库, 前端基于 Bootstrap 响应式设计, 能在多端具有良好的显示效果.

View on GitHub Download .zip Download .tar.gz

核心功能

  • 文章/图片/视频发布、喜欢、统计阅读次数。
  • 文章标签tag功能、私密文章支持、文章归档
  • 文章支持uditor/markdown编辑器切换（后台配置）
  • 评论功能，评论支持回复。
  • 会员注册登录，后台管理功能。
  • lucene实现的站内搜索。
  • 响应式布局支持(pc/pad/phone)

技术选型

  • JDK8
  • 数据库MySQL
  • 主框架 (Spring-boot、Spring-data-jpa）
  • 安全权限 Shiro
  • 搜索工具 Lucene
  • 缓存 Ehcache
  • 视图模板 Freemarker
  • Bootstrap 前端框架

支持浏览器

  • chrome/firefox/ie9+

Mblog

mblog(mtons blog)是一个用Java实现的多人轻博客. 使用Mysql数据库, 前端基于 Bootstrap 响应式设计, 能在多端具有良好的显示效果.

View on GitHub Download .zip Download .tar.gz

技术选型

  • JDK8
  • 数据库MySQL
  • 主框架 (Spring-boot、Spring-data-jpa）
  • 安全权限 Shiro
  • 搜索工具 Lucene
  • 缓存 Ehcache
  • 视图模板 Freemarker
  • Bootstrap 前端框架

支持浏览器

  • chrome/firefox/ie9+
